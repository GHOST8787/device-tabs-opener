一鍵批次開啟 iPhone / iPad 同步到 Chrome 的所有分頁。

🔗 **[前往 Chrome 線上應用程式商店安裝](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

---

## ✨ 功能

- 顯示所有已同步裝置及其分頁
- 勾選想開啟的分頁，一鍵批次開啟
- 全選 / 取消全選
- 在新視窗中開啟所有選取的分頁

---

## 📥 安裝方式

### 方法一：從 Chrome 線上應用程式商店安裝（推薦）

👉 **[裝置分頁批次開啟器 — Chrome Web Store](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

點擊「加到 Chrome」即可安裝，無需開發者模式。

### 方法二：手動安裝（開發者模式）

1. 下載或 clone 此專案
2. 開啟 Chrome，前往 `chrome://extensions`
3. 右上角開啟「**開發人員模式**」
4. 點擊「**載入未封裝項目**」→ 選擇此專案資料夾
5. 完成！工具列會出現紫色手機圖示

---

## 🚀 使用方式

1. 確認 Chrome 已登入 Google 帳號，且 iPhone / iPad 上的 Chrome 有開啟同步
2. 點擊工具列的 📱 圖示
3. 勾選想開啟的分頁（或點「全選」）
4. 點擊「開啟選取的分頁」→ 所有頁面會在新視窗中開啟

---

## 🔒 權限說明

| 權限 | 用途 |
|---|---|
| `sessions` | 讀取其他裝置同步的分頁 |
| `tabs` | 開啟新分頁 |

本擴充功能不會收集任何使用者資料。

---

## 📋 常見問題

**Q：找不到裝置分頁？**

- 確認 Chrome 已登入 Google 帳號
- 手機／平板上的 Chrome 需開啟「同步」功能
- 設定路徑：設定 → Google → 同步 → 分頁

**Q：同步了但數量不對？**

Chrome 的裝置同步需要幾分鐘才會把資料送到 Google 伺服器。可先在手機上手動開幾個分頁，等 5 分鐘後再試。
