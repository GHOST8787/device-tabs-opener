一鍵批次開啟 iPhone / iPad 同步到 Chrome 的所有分頁。

A Chrome extension to batch-open tabs synced from your iPhone / iPad.

🔗 **[前往 Chrome 線上應用程式商店安裝 / Install from Chrome Web Store](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

---

## ✨ 功能 / Features

- 顯示所有已同步裝置及其分頁 / View all synced devices and their tabs
- 勾選想開啟的分頁，一鍵批次開啟 / Select and batch-open tabs with one click
- 全選 / 取消全選 / Select all / Deselect all
- 在新視窗中開啟所有選取的分頁 / Opens selected tabs in a new window

---

## 📥 安裝方式 / Installation

### 方法一：從 Chrome 線上應用程式商店安裝（推薦 / Recommended）

👉 **[裝置分頁批次開啟器 — Chrome Web Store](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

點擊「加到 Chrome」即可安裝，無需開發者模式。
Click "Add to Chrome" to install directly — no developer mode required.

### 方法二：手動安裝（開發者模式 / Developer Mode）

1. 下載或 clone 此專案
2. 開啟 Chrome，前往 `chrome://extensions`
3. 右上角開啟「**開發人員模式**」
4. 點擊「**載入未封裝項目**」→ 選擇此專案資料夾
5. 完成！工具列會出現紫色手機圖示

---

## 🚀 使用方式 / Usage

1. 確認 Chrome 已登入 Google 帳號，且 iPhone/iPad 上的 Chrome 有開啟同步
2. 點擊工具列的 📱 圖示
3. 勾選想開啟的分頁（或點「全選」）
4. 點擊「開啟選取的分頁」→ 所有頁面會在新視窗中開啟

---

## 🔒 權限說明 / Permissions

| 權限 Permission | 用途 Purpose |
|---|---|
| `sessions` | 讀取其他裝置同步的分頁 / Read tabs synced from other devices |
| `tabs` | 開啟新分頁 / Open new tabs |

本擴充功能不會收集任何使用者資料。
This extension does not collect any user data.

---

## 📋 常見問題 / FAQ

**Q: 找不到裝置分頁？**
- 確認 Chrome 已登入 Google 帳號
- 手機/平板上的 Chrome 需開啟「同步」功能
- Settings → Google → 同步 → 分頁

**Q: Can't find device tabs?**
- Make sure you're signed in to Chrome with your Google account
- Enable "Sync" in Chrome on your iPhone/iPad
- Settings → Google → Sync → Tabs
