One-click batch opening of all tabs synced from iPhone / iPad to Chrome.

🔗 **[Install from the Chrome Web Store](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

---

## ✨ Features

- Show all synced devices and their tabs
- Check the tabs you want to open and batch-open them with one click
- Each device can "select all / deselect" separately
- Show each device's last sync time
- Open all selected tabs in a new window
- Chinese/English interface switching (automatically follows the browser language, can also be switched manually)

## 📥 Installation

### Method 1: Install from the Chrome Web Store (recommended)

👉 **[Device Tabs Batch Opener — Chrome Web Store](https://chromewebstore.google.com/detail/idmlpeiehifpmfbgpefcdjlbdchjmjpc)**

Click "Add to Chrome" to install, no developer mode needed.

### Method 2: Manual installation (developer mode)

1. Download or clone this project
2. Open Chrome and go to `chrome://extensions`
3. Turn on "**Developer mode**" in the top-right corner
4. Click "**Load unpacked**" → select this project folder
5. Done! A purple phone icon will appear in the toolbar

---

## 🚀 Usage

1. Make sure Chrome is signed in to a Google account and sync is enabled in Chrome on your iPhone / iPad
2. Click the 📱 icon in the toolbar
3. Check the tabs you want to open (or click "Select all")
4. Click "Open selected tabs" → all pages will open in a new window

## 🔒 Permissions

| Permission | Purpose |
|---|---|
| `sessions` | Read tabs synced from other devices |
| `tabs` | Open new tabs |

This extension does not collect any user data.

---

## 📋 FAQ

**Q: Can't find device tabs?**

- Make sure Chrome is signed in to a Google account
- Chrome on your phone/tablet must have "Sync" enabled
- Settings path: Settings → Google → Sync → Tabs

**Q: Synced but the count is wrong?**

Chrome's device sync takes a few minutes to send data to Google's servers. Try manually opening a few tabs on your phone, then retry after 5 minutes.
